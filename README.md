# MD 內容儲存器

這是一個可以放到 GitHub Pages 的純靜態 Markdown 小工具。

## 功能

- 編輯 Markdown 內容
- 即時預覽標題、清單、引用、程式碼區塊、粗體、斜體和連結
- 把草稿存在瀏覽器 localStorage
- 下載目前內容成 `.md` 檔案

## 本機使用

直接用瀏覽器開啟 `index.html`。

## 推到 GitHub Pages

1. 在 GitHub 建立一個新 repository。
2. 把這個資料夾 push 到該 repository。
3. 到 repository 的 Settings > Pages。
4. Source 選 `Deploy from a branch`，Branch 選 `master` 或 `main`，資料夾選 `/root`。
5. 儲存後等待 GitHub 產生網址。
