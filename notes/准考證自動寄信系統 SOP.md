# 准考證自動寄信系統 SOP

> 115年度原住民族語言能力認證模擬測驗  
> 建立日期：2026-06-26  
> 適用對象：負責發送准考證通知的行政人員

---

## 為什麼要用這套系統？

### 傳統手動寄信的問題

- 每封信要手動填姓名、場次、時間 → **容易打錯**
- 每封附件要一張一張找圖片 → **耗時又容易附錯**
- 寄了幾封、哪些失敗 → **沒有記錄**
- 下次再辦，整個流程要重來 → **無法重複使用**

### 這套系統解決了什麼

| 問題 | 解法 |
|------|------|
| 手動填資料容易出錯 | 直接從 Google Sheet 讀取，零手動輸入 |
| 附件找錯人 | 自動依「姓名_級別_場次」比對，附對的圖 |
| 不知道寄了沒 | Sheet J欄自動標記「已寄出」，重跑不重複 |
| 流程不可重複 | 架構標準化，下次換資料就能直接用 |
| 找不到圖片仍要寄 | 找不到圖片照樣寄，信末加紅字提醒，不中斷流程 |

## 系統架構

```
Google Sheet（考生資料）
       ↓
  Apps Script
  ├── Code.gs      後端：讀資料、找圖片、寄信
  └── Sidebar.html 前端：操作介面（側欄）
       ↓
Google Drive（准考證圖片資料夾）
       ↓
Gmail → 考生信箱
```

## 今天的完整工作流程

### Phase 1｜整理圖片

**問題：** 圖片原始檔名是 `未命名-1007000-02.png`，無法對應考生。

**解法：** 逐張辨識圖片內容，依規則重新命名。

**命名規則：**
```
姓名_級別_場次.png

範例：
簡伶雯_中高級_第四場.png
吳岱衡_初級_第三場.png
莎冷．夫乎夫克_中高級_第六場.png
```

**注意事項：**
- 場次只寫「第X場」，不含時間
- 姓名與圖片內容完全一致，包含特殊符號（如全形「．」）
- 同一人報名多個級別 → 每個級別各一張獨立圖片
- 同一張圖片含多個場次（如吳岱衡）→ 需另外製作分開的圖，或暫時略過

---

### Phase 2｜建立 Apps Script

**兩個檔案：**

| 檔案 | 用途 |
|------|------|
| `Code.gs` | 後端邏輯：讀 Sheet、比對 Drive 圖片、寄 Gmail |
| `Sidebar.html` | 前端介面：輸入網址、預覽清單、觸發寄送 |

**部署步驟：**

1. 開啟 Google Sheet → 擴充功能 → Apps Script
2. 預設 `Code.gs` 貼上後端程式碼
3. 點「+」新增 HTML 檔，命名 `Sidebar`，貼上前端程式碼
4. 存檔（Ctrl+S）
5. 執行 `onOpen` → 完成 Google 授權（只需做一次）
6. 回 Sheet 重新整理 → 出現「📧 寄信工具」選單

---

### Phase 3｜操作介面三步驟

```
① 設定
   └─ 貼上 Google Drive 准考證圖片資料夾網址
   └─ 點「載入考生清單」

② 預覽
   └─ 確認每位考生的圖片狀態（✓ 有 / ✗ 無）
   └─ 確認場次、級別對應正確
   └─ 點「前往寄送」

③ 寄送
   └─ 點「開始寄送」
   └─ 即時看每封進度與 log
   └─ 完成後 Sheet J欄自動標記「已寄出」
```

## 使用時機

### 適合用這套系統的情況

- **批次寄送帶附件的個人化通知**（准考證、錄取通知、成績單）
- **每人附件不同**，需要自動比對對應檔案
- **需要防止重複寄送**（系統中途中斷後重跑）
- **資料已在 Google Sheet**，不想另外建資料庫
- **非工程師操作**，需要有介面點按執行

### 不適合用這套系統的情況

- 收件人超過 500 人（Gmail 免費版每日上限約 100 封，G Suite 約 1500 封）
- 需要追蹤「開信率」「點擊率」等行銷分析數據 → 改用 Mailchimp、Brevo 等工具
- 附件是動態產生的（如個人化 PDF）→ 需額外加入 PDF 生成模組

## 圖片命名 SOP（可重複使用）

每次新的活動，只要依照這個規則命名，系統就能直接對應：

```
【規則】
姓名_級別_場次.png

【級別選項】
初級 / 中級 / 中高級 / 高級

【場次格式】
第一場 / 第二場 / 第三場 ... （不加時間）

【特殊情況處理】
同人多報名  → 各自一張：吳岱衡_初級_第三場.png、吳岱衡_中級_第四場.png
原住民名字  → 保留全形符號：莎冷．夫乎夫克_中高級_第六場.png
一圖多場次  → 需拆成獨立圖片，或在 Drive 中複製成多份
```

## 已知問題與解法

| 問題 | 原因 | 解法 |
|------|------|------|
| 圖片顯示「✗ 無」 | 檔名不符合規則 | 重新檢查命名，特別注意全形符號 |
| 寄送失敗 | Gmail 頻率限制或 token 過期 | 直接重跑，已寄的會跳過 |
| 選單不出現 | 未重新整理 Sheet | 按 F5 重新整理 |
| 授權視窗一直跳 | 未完成授權流程 | 回 Apps Script 重新執行 `onOpen` |
| 吳岱衡只有一張圖含兩場次 | 原始圖片設計問題 | 等新圖或複製成兩份不同檔名 |
